# Emotion Detector Application

This project is an AI-based web application that detects human emotions from text input using IBM Watson NLP.

## 📌 Features

- Detects emotions such as:
  - Joy 😊
  - Anger 😠
  - Disgust 🤢
  - Fear 😨
  - Sadness 😢
- Uses IBM Watson NLP API for emotion analysis
- Displays formatted output of detected emotions
- Web interface built using Flask
- Includes error handling for invalid input
- Unit tested and validated

## 🛠️ Technologies Used

- Python
- Flask
- IBM Watson NLP
- Requests Library
- Pylint (for code analysis)
